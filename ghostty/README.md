# Ghostty 操作マニュアル

## 設定ファイル（config）の概要

| 項目 | 設定値 |
|------|--------|
| フォントサイズ | `12` |
| テーマ | Bluloco Dark |
| カーソル | バースタイル（点滅なし） |
| ステータス保存 | 常時（再起動時に復元） |

---

## ペイン操作

### 分割

| 操作 | キーバインド |
|------|-------------|
| 右に分割（縦斬り） | `Ctrl + Shift + h` |
| 下に分割（横斬り） | `Ctrl + Shift + n` |

### 移動

| 操作 | キーバインド |
|------|-------------|
| 上のペインへ | `Ctrl + Shift + i` |
| 左のペインへ | `Ctrl + Shift + j` |
| 下のペインへ | `Ctrl + Shift + k` |
| 右のペインへ | `Ctrl + Shift + l` |

### 閉じる

| 操作 | キーバインド |
|------|-------------|
| 現在のペインを閉じる | `Ctrl + Shift + x` |

---

## タブ操作（デフォルトキーバインド）

| 操作 | キーバインド |
|------|-------------|
| 新しいタブを開く | `Ctrl + Shift + t` |
| 次のタブへ | `Ctrl + Shift + ]` |
| 前のタブへ | `Ctrl + Shift + [` |
| タブを閉じる | `Ctrl + Shift + w` |

---

## その他（デフォルトキーバインド）

| 操作 | キーバインド |
|------|-------------|
| 新しいウィンドウを開く | `Ctrl + Shift + n`（分割なしの場合） |
| フォントサイズを拡大 | `Ctrl + =` |
| フォントサイズを縮小 | `Ctrl + -` |
| フォントサイズをリセット | `Ctrl + 0` |
| 設定をリロード | `Ctrl + Shift + ,` |
