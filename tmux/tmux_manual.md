# tmux 操作マニュアル

## 設定ファイル（.tmux.conf）の概要

| 項目 | 設定値 |
|------|--------|
| Prefix キー | `Ctrl + s` |
| シェル | `zsh` |
| ウィンドウ開始番号 | `1` |
| テーマ | Catppuccin Mocha |
| ステータスバー | 上部に配置 |

---

## 基本操作

Prefix = `Ctrl + s`

### 画面分割・ペイン操作

| 操作 | コマンド |
|------|----------|
| 左右に分割（縦斬り） | `Prefix` + `h` |
| 上下に分割（横斬り） | `Prefix` + `n` |
| ペイン移動（上） | `Prefix` + `i` |
| ペイン移動（左） | `Prefix` + `j` |
| ペイン移動（下） | `Prefix` + `k` |
| ペイン移動（右） | `Prefix` + `l` |
| ペインを閉じる | `exit` または `Ctrl + d` |

### デタッチ・アタッチ

| 操作 | コマンド |
|------|----------|
| デタッチ（バックグラウンドへ） | `Prefix` + `d` |
| 直前のセッションに復帰 | `tmux a` |

### セッション管理

| 操作 | コマンド |
|------|----------|
| 名前付きで起動 | `tmux new -s <name>` |
| セッション指定でアタッチ | `tmux a -t <name>` |
| セッション一覧 | `tmux ls` |
| セッション名を変更 | `Prefix` + `$` |
| セッションを終了 | `tmux kill-session -t <name>` |

### 名前変更

| 操作 | コマンド |
|------|----------|
| ウィンドウ名を変更 | `Prefix` + `,` |

### 設定のリロード

| 操作 | コマンド |
|------|----------|
| `.tmux.conf` を再読み込み | `Prefix` + `r` |

---

## バックグラウンド処理の手順

```
tmux new -s <session-name>   # セッション開始
python train.py              # 処理を実行
Ctrl+s → d                   # デタッチ（処理は継続）
# ターミナル・PCを閉じてよい
tmux a -t <session-name>     # 後から復帰して結果確認
```
