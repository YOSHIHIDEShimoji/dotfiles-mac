# tmux 操作マニュアル

## 設定ファイル（.tmux.conf）の概要

| 項目 | 設定値 |
|------|--------|
| Prefix キー | `Ctrl + s` |
| シェル | `zsh` |
| ウィンドウ開始番号 | `1` |
| テーマ | Catppuccin Mocha |
| ステータスバー | 上部に配置 |

---

## 基本操作

Prefix = `Ctrl + s`

### 画面分割・ペイン操作

| 操作 | コマンド |
|------|----------|
| 左右に分割（縦斬り） | `Prefix` + `h` |
| 上下に分割（横斬り） | `Prefix` + `n` |
| ペイン移動（上/左/下/右） | `Prefix` + `i` / `j` / `k` / `l` |
| ペイン番号を表示・移動 | `Prefix` + `q`（番号を入力で移動） |
| ペインを最大化 / 元に戻す | `Prefix` + `z` |
| ペインを閉じる | `exit` または `Ctrl + d` |

### ウィンドウ操作

| 操作 | コマンド |
|------|----------|
| 新しいウィンドウを作成 | `Prefix` + `c` |
| 次のウィンドウへ | `Prefix` + `n` |
| 前のウィンドウへ | `Prefix` + `p` |
| 番号指定で移動 | `Prefix` + `1`〜`9` |

### 名前変更

| 操作 | コマンド |
|------|----------|
| ウィンドウ名を変更 | `Prefix` + `,` |
| セッション名を変更 | `Prefix` + `$` |

### コピーモード

| 操作 | コマンド |
|------|----------|
| コピーモード開始 | `Prefix` + `[` |
| ページスクロール（上） | `Ctrl + b` |
| ページスクロール（下） | `Ctrl + f` |
| モード終了 | `Enter` または `q` |
| マウス操作 | ホイールでスクロール、ドラッグで選択 |

### セッション管理

| 操作 | コマンド |
|------|----------|
| デタッチ（バックグラウンドへ） | `Prefix` + `d` |
| 直前のセッションに復帰 | `tmux a` |
| セッション一覧 | `tmux ls` |
| 名前付きで起動 | `tmux new -s <n>` |
| セッション指定でアタッチ | `tmux a -t <n>` |
| セッションを終了 | `tmux kill-session -t <n>` |

### 設定のリロード

| 操作 | コマンド |
|------|----------|
| `.tmux.conf` を再読み込み | `Prefix` + `r` |

---

## バックグラウンド処理の手順

```
tmux new -s <session-name>   # セッション開始
python train.py              # 処理を実行
Ctrl+s → d                   # デタッチ（処理は継続）
# ターミナル・PCを閉じてよい
tmux a -t <session-name>     # 後から復帰して結果確認
```
